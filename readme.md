## Menjalankan Situs Secara Lokal

Jika Anda ingin menjalankan situs ini di mesin lokal (untuk mengedit konten atau melihat perubahan):

1. Pastikan `Hugo Extended`, `Node.js`, dan `npm`/`yarn` sudah terpasang.
2. Install dependency aset (jika perlu):

```bash
npm install
```

3. Jalankan server development:

```bash
npm run dev
```

Atau gunakan task workspace yang tersedia: `yarn dev:example`.
